# Chrome Store Assets Design

## Goal

Create Chrome Web Store screenshots and promotional images for JSON Formatter with mixed Chinese and English copy.

## Approved Direction

Use polished, deterministic store visuals based on the extension's real product surface. The images should show the formatter as a developer tool: readable JSON, syntax colors, line numbers, collapse controls, copy actions, and local privacy messaging.

## Asset Set

- Five screenshots at 1280x800.
- One small promotional image at 440x280.
- One marquee promotional image at 1400x560.

## Visual System

- Clean developer-tool layout with a browser-like frame.
- Light interface panels with dark JSON code areas where it improves contrast.
- Mixed copy: English feature headline plus Chinese explanation.
- Keep promo images simple and brand-led, with limited text.

## Screenshot Concepts

1. Auto format raw JSON / 自动格式化页面 JSON.
2. Collapse complex data / 折叠复杂对象与数组.
3. Line numbers and syntax highlight / 行号与语法高亮.
4. Paste formatter page / 粘贴后实时格式化.
5. Local and fast / 本地处理，不上传数据.

## Source Requirements

Chrome Web Store supports screenshots at 1280x800 or 640x400, with 1280x800 preferred. The required small promotional image is 440x280, and the optional marquee image is 1400x560.
