# 🎨 JSON Formatter 使用指南：让数据清晰易读

**JSON Formatter** 是一款专为 Chrome 浏览器设计的“美容”工具。它可以把网页上那些密密麻麻、乱成一团的 JSON 代码，瞬间变成像文件夹一样整齐、美观且易于阅读的界面。

---

## 🚀 第一步：安装扩展（只需 3 分钟）

由于本工具是“开发者版本”，安装方式比普通插件稍微多一个小步骤：

1. **准备文件**：将下载好的项目文件夹保存在电脑上（记住位置，不要删除它）。
2. **打开扩展页面**：在 Chrome 浏览器地址栏输入 `chrome://extensions` 并回车。
3. **开启“开发者模式”**：点击页面右上角的开关，确保它处于**开启**状态。
4. **载入工具**：点击左上角的 **“加载已解压的扩展程序”** 按钮。
5. **选择文件夹**：在弹出的窗口中，选择你刚刚下载的那个文件夹，点击“选择”。

> **💡 小贴士：** 看到页面上出现了“JSON Formatter”的卡片，就说明你成功了！

---

## 🖱️ 第二步：如何使用

安装完成后，你不需要进行任何操作，它会自动在后台为你工作。

### 1. 自动美化

当你用 Chrome 打开一个 JSON 网页（通常是一堆黑白文字的页面）时，你会发现页面变了：

* **彩色文字**：不同的数据类型（数字、文字、链接）会有不同的颜色。
* **行号显示**：左侧会出现数字，方便你数行数。
* **自动缩进**：内容不再挤在一起，而是像大纲一样整齐。

### 2. 常用功能按钮

在页面的右上角，你会看到几个实用的控制按钮：

* **展开/折叠全部**：一键收起所有内容，或者全部展示。
* **折叠 1级/2级/3级**：像剥洋葱一样，只看最外层或者更深层的数据。
* **复制 JSON**：一键把整理好的内容存入剪贴板，粘贴到别处时依然保持整齐。

### 3. 手动操作

* **点点看**：每个方括号 `[` 或大括号 `{` 左边都有一个小减号 `-`，点击它可以把这一块内容藏起来。

## 🛠️ 进阶小技巧

### 如果要查看电脑本地的 JSON 文件

如果你双击电脑里的 `.json` 文件却没反应，请按以下操作：

1. 回到 `chrome://extensions` 页面。
2. 点击 JSON Formatter 卡片上的 **“详细信息”**。
3. 向下滑动，找到 **“允许访问文件网址”** 并勾选。
4. 现在，直接把电脑里的 JSON 文件拖进 Chrome 就能看了。

### 遇到超大型文件怎么办？

如果数据量特别巨大（比如几万行），为了不让浏览器卡死，工具会切换到 **“纯文本模式”**：

* 此时只会显示文字，没有彩色和折叠功能。
* 如果你一定要看美化版的，可以点击页面上的 **“启用折叠”** 按钮，但请耐心等待加载。

---

## ❓ 常见问题

* **为什么有的页面没有变美？**
工具很智能，它只会在“纯 JSON 数据”页面运行。如果页面本身包含图片、广告或复杂的网页设计，它会自动回避以防弄坏网页。
* **我的数据安全吗？**
非常安全。该工具完全在你的电脑本地运行，**不会**把任何数据上传到互联网。
